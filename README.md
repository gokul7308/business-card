# 💼 Business Card Website

A simple and elegant personal **business card web page** built using HTML and CSS. This project showcases a clean UI design with an "About Me" section and profile image.

- ✨ Minimal and modern design
- 📱 Responsive layout
- 👤 Personal introduction section
- 🖼️ Profile image display
- 🎨 Styled using pure CSS
